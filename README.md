# 📚 Book Store

A simple and functional Book Store web application where users can **create**, **read**, **update**, and **delete** (CRUD) books. This project is built to demonstrate full-stack development with a clean and minimal UI/UX.

## 🚀 Features

- Add new books with title, author, genre, and description
- View the list of all books
- Update book details
- Delete books from the collection
- Responsive design for all devices
- User-friendly interface

## 🛠️ Tech Stack

**Frontend:**
-  React, Tailwind CSS

**Backend:**
- Node.js, Express.js

**Database:**
- MongoDB (or specify if you're using SQL/another NoSQL database)
